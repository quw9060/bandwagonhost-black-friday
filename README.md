## 搬瓦工是什么，为什么每年黑五值得关注

搬瓦工（BandwagonHost）是加拿大 IT7 Networks 旗下的 VPS 服务商，做了二十多年了。它在国内用户里出名，主要靠两件事：一是早年一堆低价套餐让很多人第一次玩起了 VPS，二是后来布局 CN2 GIA 高速线路之后，成了国内访问国外服务器最稳定的选项之一。

黑色星期五（Black Friday）是北美每年感恩节后第一个周五，时间大概在 11 月下旬，是老外版的"双十一"。搬瓦工历史上在黑五出过不少大力度折扣，所以每年到了这个时间节点，关注搬瓦工的人就开始蠢蠢欲动。

## 为什么搬瓦工近几年不搞大黑五了

这个问题值得展开说说，不然每年等到失望是正常的。

搬瓦工这几年的策略变了。早年走的是低价大众路线，靠着 $19.99、$29.99 的年付套餐拉用户，黑五大促是很自然的动作。但从 2021 年前后，它开始把产品线往中高端推：下架了大部分低配便宜套餐，重点经营 CN2 GIA-E、香港 CN2 GIA、东京 CN2 GIA 这些高端线路。

走高端路线的商家，通常不靠打折维持声量。搬瓦工现在更倾向于不定期推出"限量版套餐"——比如 THE PLAN 系列、DC9 限量版、MINICHICKEN 这类——给老用户传家宝级别的折扣，而不是全场打折拉新。这些限量版套餐往往比常规套餐便宜一半甚至更多，但库存极少，补货了几小时内就没了。

所以说，等搬瓦工黑五大促，不如关注限量版套餐补货。

以前搬瓦工有一批可以长期使用的循环折扣码，比如 BWHCGLUKKB（6.77% 折扣），在结账时填入，首购和续费都有效。
